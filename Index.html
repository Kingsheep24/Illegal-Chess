Introduction

(please close this page if you aren't involved with the development) Infinity Fusion Chess is based off international chess, with the biggest change being the addition of crafting and promoting. Other additions such as effects, health / damage system and three-dimentional movement (air, ground and void) make the game seemingly unpredictable at times. This is to create a fun atmosphere for both beginners and pros at chess as everyone has to learn the rules again from scratch. This game is non-competitive and created purely for entertainment purposes, so have fun playing!

you can check out the "how to play" page above if you are interested in playing the alpha version (not out yet). Feel free to leave suggestions on how to balance or improve the game!
